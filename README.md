# Student Management System

A full-stack web application to manage student records, attendance, academic results, and report cards.

## Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** Node.js, Express.js
- **Database:** MySQL (Sequelize ORM)

## Features

- Add, edit, delete student records
- Class management
- Attendance tracking
- Result management with auto grade calculation
- Student profile with performance overview
- Printable report cards
- Search, sort, and pagination

## Pages

- Dashboard — student stats and quick actions
- Students — CRUD operations
- Profile — individual student overview
- Attendance — mark and view attendance
- Results — add and manage academic results
- Report Card — printable academic report

## Run Locally

**Backend:**
```bash
cd student-management
npm install
node server.js
```

**Frontend:**
Open `frontend/index.html` in your browser
